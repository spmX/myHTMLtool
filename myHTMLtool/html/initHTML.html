initHTML {myHTMLtool}R Documentation

initate an html object

Description

create an new html object and set up folders containing css and js libraries

Usage

initHTML(title = "htmlTitle", outDir = ".", outPrefix = "htmlReport")

Arguments

outDir

path The path to store output files

outPrefix

character The string used for creating output folder

titile

character The title that goes to the <titile/> tag

Value

an html object
